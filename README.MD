Simple API for experiments. Project link: [http://rg-rest-api.herokuapp.com](http://rg-rest-api.herokuapp.com)

### Tech Stack
- express
- mongoose
- mongoDB
- jade
- morgan
- yarn
- heroku


#### API

```
// get all posts
GET /api/posts/
```
```
// create a new post
POST /api/posts/
```
```
// get post by ID
GET /api/posts/:id
```
```
// update post by ID
PUT /api/posts/:id
```
```
// delete post by ID
DELETE /api/posts/:id
```
